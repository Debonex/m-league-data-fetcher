## About

Fetch m-league paifu information.

## Usage

- sign in [https://m-league.jp/](https://m-league.jp/)

- "日程・対戦成績" => choose any match day => "牌譜をみる"

- check site cookies, copy paifuSessionId and enter it in the .env file.

-

```shell
npm install
npm start
```

or

```shell
yarn
yarn start
```

## Other
