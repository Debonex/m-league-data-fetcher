## About

Fetch and store m-league haifu(牌譜) information.

## Usage

```shell
npm install
```

### fetch haifu information

```shell
npm run fetch
```

### store haifu information to database (sqlite3)

init your database with `sql/init.sql`

```shell
npm run store
```

### Other

Tested environment:

- node: v16.14.0
